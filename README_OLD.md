# OpenClaw Trading System Backup
Este repositorio contiene el sistema completo de trading algorítmico con Swarm AI.

## Estructura:
- strategies/: Agentes neuronales y estrategias de trading
- scripts/: Scripts Python para ejecución y monitoreo
- configs/: Archivos de configuración
- results/: Resultados de pruebas y métricas
- docs/: Documentación y guías
- backups/: Scripts de backup automático
